attr()方法设置或返回被选元素的属性
$(selector).attr(attribute)

remove attr()方法删除特定的属性

二关于jQuery中html()，text()，val()的使用

html()方法当使用该方法返回一个值时，它会返回第一个匹配元素的内容
$(selector).html(content) 使用函数来设置所有匹配元素的内容。
$(selector).html(function(index,oldcontent))
index - 可选。接收选择器的 index 位置。
oldcontent - 可选。接收选择器的当前内容。

$(A).append(B)将B追加到A当中

$(需要添加的内容).appendTo("showDiv")将需要添加的内容加到showDiv中

三empty和remove的区别

empty是将指定元素中子元素(包括文本节点)删除,remove是从DOM中删除，而不会保留其所占的位置

四关于jQuery中CSS的设置

$().css(name,value)中
中name是 必需。规定 CSS 属性的名称。该参数可包含任何 CSS 属性，比如 "color"。
中value是 可选。规定 CSS 属性的值。该参数可包含任何 CSS 属性值，比如 "red"。 如果设置了空字符串值，则从元素中删除指定属性。

$().css(name,function(index,value))中
index - 可选。接受选择器的 index 位置
oldvalue - 可选。接受 CSS 属性的当前值 设置多个值时用大括号隔开

window.onscroll指的是滚动条事件
